# Banker's Algorithm Simulator 🏦💻

## 📌 Project Overview
An interactive, web-based simulator designed to visualize and execute the **Banker's Algorithm**, a core Operating Systems concept used for deadlock avoidance and safe resource allocation. 

This educational tool allows users to input custom processes and resources, defining 'Available', 'Max', and 'Allocation' matrices, to step-by-step determine if the system is in a safe state and output the safe sequence.

## ⚙️ Key Features
* **Interactive Matrix Inputs:** Dynamically adjust the number of processes and resources. The UI automatically generates the appropriate grid inputs for the matrices.
* **Step-by-Step Visualization:** Clearly displays the mathematical calculations and state changes, making it an excellent educational resource for understanding deadlocks.
* **Pre-loaded Examples:** Includes built-in test cases to quickly demonstrate how the algorithm functions under different intensive scenarios.
* **Modern UI/UX:** Features a clean, responsive design with a sleek **Dark/Light Mode** toggle for optimal user experience.

## 🛠️ Tech Stack
* **Frontend Structure:** HTML5
* **Styling:** CSS3 (Custom Variables, Flexbox/Grid, Dark Mode Implementation)
* **Logic & Interactivity:** Vanilla JavaScript (DOM Manipulation, Matrix Calculations, State Management)

## 📂 Repository Structure
* `index.html`: The main structural entry point of the simulator.
* `styles.css`: Contains all styling, including responsive media queries and dark mode themes.
* `script.js`: The core logic engine handling the Banker's algorithm calculations and UI updates.
* `flowchart.jpg`: A visual flowchart detailing the algorithm's decision-making process.
